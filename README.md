## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/qamro/blue-team-roadmap
cd blue-team-roadmap

# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
